# UnityMicrophonePractice

<div style="text-align:right">Language: <a href="README.md">English</a> | <i>日本語</i></div>

Microphoneクラスで正確な録音時間を取得するためのサンプルプロジェクトです。



詳細はこちらをご覧ください。
https://qiita.com/segur/items/6d7b140418ef9adf6e7c



## 使用方法

ボタンは３つのみです。

- Start Record：録音開始
- Stop Record：録音停止
- Play：再生
